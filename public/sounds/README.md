# 사운드 파일 안내

이 폴더에 OGG 형식의 사운드 파일을 추가하세요.

## 필요한 사운드 파일

### 효과음
- `upgrade_success.ogg` - 강화 성공 사운드
- `upgrade_fail.ogg` - 강화 실패 사운드
- `click.ogg` - 클릭 사운드
- `kill.ogg` - 킬 사운드
- `tower.ogg` - 타워 파괴 사운드
- `dragon.ogg` - 드래곤 처치 사운드
- `baron.ogg` - 바론 처치 사운드

### 브금
- `main_bgm.ogg` - 메인 화면 브금
- `match_bgm.ogg` - 경기 중 브금
- `victory_bgm.ogg` - 승리 브금
- `defeat_bgm.ogg` - 패배 브금

## 파일 형식
- 모든 사운드 파일은 **OGG 형식**이어야 합니다.
- 파일 크기는 가능한 작게 유지하세요 (각 파일 1MB 이하 권장).

## 사용 방법
1. OGG 형식의 사운드 파일을 이 폴더에 추가하세요.
2. 파일 이름은 위의 목록과 정확히 일치해야 합니다.
3. 서버를 재시작하면 자동으로 로드됩니다.
